# 仿爱彼迎官网

概要: 这是一个仿爱彼迎官网的静态首页

在线部署地址：[点击查看ヾ(=･ω･=)o](https://rayadaschn.github.io/airbnb-web)

后端接口有 APIFox 设置

---

### 项目安装依赖

```bash
npm install
```

### 测试运行

```bash
npm run start
```
